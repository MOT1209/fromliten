# Rust Game v6.0 - Advanced Building System

## النظام الجديد: البناء المتقدم (Rust-Inspired)

### الميزات الأساسية:

#### 1. **Blueprint Selector (اختيار المخطط)**
- اضغط **Q** لفتح قائمة اختيار البناء
- أنواع البناء المتاحة:
  - **Foundation** (الأساس): قاعدة البناء
  - **Wall** (الجدار): حماية عمودية
  - **Doorway** (مدخل الباب): نقطة دخول
  - **Ceiling** (السقف): حماية علوية
  
#### 2. **Building Tiers (مستويات البناء)**
يبدأ كل بناء كـ **Twig** (عيدان) ويمكن ترقيته تدريجياً:

| المستوى | الصحة | المواد المطلوبة | اللون |
|---------|------|-----------------|-------|
| **Twig** | 10 | مجاني | بيج فاتح |
| **Wood** | 250 | 300 خشب | بني |
| **Stone** | 500 | 300 حجر | رمادي |
| **Sheet Metal** | 1000 | 200 معدن | رمادي معدني |
| **Armored (HQM)** | 2000 | 50 معدن عالي الجودة | فضي |

#### 3. **الوضع والبناء**
- اختر مخططاً من القائمة
- انقر بزر الفأرة الأيسر لوضع القطعة
- **Snap to Grid**: القطع تنجذب لشبكة 3×3 للمحاذاة المثالية
  
#### 4. **الترقية والإصلاح**
- **اضغط H** عند النظر للبناء:
  - إذا كان مكتمل الصحة → **ترقية** للمستوى التالي
  - إذا كان تالفاً → **إصلاح** (يستهلك 10% من تكلفة الترقية)
  
#### 5. **Build Info Panel (لوحة معلومات البناء)**
عند النظر لأي بناء تظهر:
- **Tier**: المستوى الحالي
- **Health**: الصحة الحالية / الحد الأقصى  
- **Upgrade Cost**: تكلفة الترقية للمستوى التالي

#### 6. **Tool Cupboard (TC) Protection**
- النظام جاهز لإضافة **Tool Cupboard**
- يمنع البناء داخل منطقة TC للاعبين غير مصرح لهم
- رسالة "Building Blocked" تظهر عند المحاولة

### التحكم (Controls):
- **Q**: فتح/إغلاق Blueprint Selector
- **G**: إلغاء وضع البناء الحالي
- **H**: ترقية/إصلاح البناء
- **LMB**: وضع البناء المختار
- **E**: المخزون

تم إضافة موارد ابتدائية:
